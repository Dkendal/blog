How to install Neovim nightly

Posted on Jan 24, 2021

Here’s a quick little script that you can use build and install the most recent release of Neovim.

#!/bin/bash
cd ~ || exit 1
sudo rm -r neovim || true
git clone https://github.com/neovim/neovim
cd neovim || exit 1
sudo make CMAKE_BUILD_TYPE=Release install
cd ~ || exit 1
sudo rm -r neovim

I’ve used this on both Linux and Mac OSX without issue. This script was originally posted on a Neovim GitHub issue tracker which I’ve long since forgotten.
